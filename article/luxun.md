# 鲁迅（节选）

{% tabs %}
{% tab title="中英版" %}
有一分热，发一分光， \
就令萤火一般，也可以在黑暗里发一点光，不必等候炬火。 \
此后如竟没有炬火：我便是唯一的光。

One heat , light one bit ,\
Like firefly , light in dark , no need to wait .\
If nothing comes : I will be the only light .
{% endtab %}
